wlxwebkit
========
This plugin allows you to view HTML/XHTML files. It is based on webkit2gtk engine.

![plug-preview](https://i.imgur.com/2cTQkxF.png)

## Compatibility
- `doublecmd-gtk3`

## Notes
Requires version of `webkit` compiled with GTK3 support.

## Dependencies
![arch](https://wiki.archlinux.org/favicon.ico) `pacman -S webkit2gtk`
