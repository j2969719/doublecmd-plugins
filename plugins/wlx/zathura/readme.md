zathura
========
Displays PDF, DjVu, PostScipt, CBR.

![plug-preview](https://i.imgur.com/tulFpUd.png)

## Compatibility
- `doublecmd-gtk2`
- `doublecmd-gtk3`

## Notes
>zathura v2026.07.02 changelog:
>
>- Port to Gtk 4. Note that due to changes between Gtk 3 and 4, **window embedding is no longer supported**.


_so long and thanks for all the fish_

Requires `zathura` (gtk3). If Zathura has an epub plugin installed, then you can add `|EXT="EPUB"` to detectstring in `Options->Plugins->Plugins WLX->Tweak`
